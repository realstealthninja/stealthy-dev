The arduinos worked on variant of C++, i prefer to call it embeded C but that's probably Very wrong. There are 14 digital pins on an arduino board they output/input ones and zeros and 6 analog pins which input/output a range of values most likely from 0-1024 why? because the chip has processor for analog I/O operations which converts the physical electric signals into ones and zeros.

What does a arduino file look like?

Good question! An arduino is coded using a file with an extension of .ino usually named something like sketch⟨somenumber⟩.ino which has two parts a setup function and a loop function. The setup function runs the code inside only ONCE and the loop function as you've guessed it runs it in a loop until the end of time or until you power the arduino off.
