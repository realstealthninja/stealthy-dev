We sat at the very back, and the workshop had began with bang. They introduced us to the arduino showing us how it worked and doing some basic things like lighting an onboard LED of the Arduino. The arduinos worked on variant of C++, i prefer to call it embeded C but that's probably Very wrong. There are 14 digital pins on an arduino board they output/input ones and zeros and 6 analog pins which input/output a range of values most likely from 0-1024 why? because the chip has processor for analog I/O operations which converts the physical electric signals into ones and zeros. Here is the code for lighting an LED 

    
/**
 * @brief Lights the LED with the given pin number
 */
void light_led(int num){
  digitalWrite(num + 3, HIGH); 
  delay(num * 2000);
  digitalWrite(num + 3, LOW);
}
