To Python or not to Python

After a small break, I decided I wanted a change of pace; Learning a new programming language, Python, a dynamically typed, interpreted programming language. Python was easy, Python was good, and Python held your hand. It kept growing on me. I made a lot of discord bots and other unfinished projects in it. I also participated in a code jam for it, in which my team placed top 10! However, Python is an interpreted language. According to the speed nerds, it's too slow. Thus I moved on from Python as my preferred programming language.
